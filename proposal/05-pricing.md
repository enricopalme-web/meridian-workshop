# Pricing

**RFP #MC-2026-0417 — Inventory Dashboard Modernization**
Submitted by: Accenture
Date: April 28, 2026

---

We propose a **fixed-fee engagement** with milestone-based payment. You pay for outcomes, not hours. There are no hourly overruns and no change orders within the defined scope.

---

## Required Scope (R1–R4)

| Phase | Deliverables | Fee |
|---|---|---|
| Phase 1 — Onboarding & Audit | Architecture draft (R4), defect log (R1) | $8,000 |
| Phase 2 — Core Delivery | Reports remediation (R1), Restocking view (R2), first test flows (R3) | $22,000 |
| Phase 3 — Hardening & Handoff | Full test suite (R3), architecture final (R4), IT sign-off | $10,000 |
| **Total — Required Scope** | | **$40,000** |

Payment is tied to milestone acceptance, not calendar dates. If a milestone is delayed on our side, the payment date moves accordingly.

---

## Optional Scope (D1–D3)

| Item | Deliverable | Fee |
|---|---|---|
| D1 — UI Modernization | Design token refresh, component styling | $8,000 |
| D2 — Internationalization | Japanese locale (`ja`) across all views | $6,000 |
| D3 — Dark Mode | CSS theme + operator toggle | $5,000 |
| **Total — Optional Scope** | | **$19,000** |

Optional items can be elected individually or as a package. Package discount of 10% applies if all three are elected together: **$17,100**.

---

## Not-to-Exceed

The not-to-exceed for the full engagement (required + all optional) is **$57,100** ($40,000 + $17,100 with package discount).

If scope changes are requested by Meridian after contract execution, we will provide a written estimate before proceeding. We will not begin out-of-scope work without written approval.

---

## Assumptions

- Meridian provides repository access and a working development environment within two business days of contract execution.
- No hardware, infrastructure, or third-party software licenses are required or included.
- Travel is not anticipated; all work is performed remotely. If on-site sessions are requested, travel costs will be estimated and approved separately.
- Optional scope (D1–D3) pricing assumes translation materials for D2 are provided by Meridian. If Accenture is asked to produce translations, an additional $1,500 flat fee applies.
